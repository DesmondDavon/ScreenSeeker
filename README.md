🎬 Screen Seeker

Welcome to Screen Seeker!

Have you ever wanted to find a movie to watch but didn't know where to start? That's exactly why I built this project.

Screen Seeker lets you search for movies and discover what's trending right now. It's fast, simple, and fun to use.

✨ What can it do?
🎥 Search for movies by name
🔥 See what's trending
⭐ View movie ratings
📅 See when a movie was released
🖼️ Look at movie posters
🛠️ What did I build it with?

I kept things simple:

HTML
CSS
JavaScript
The Movie Database (TMDb) API
🚀 How do I use it?
Open the website.
Type the name of a movie.
Press Search.
Explore the results and discover something new to watch!

It's that easy.

💡 Why did I make this?

I built Screen Seeker to practice using APIs and JavaScript while creating something people would actually enjoy using. It helped me learn how websites can talk to other websites and display real, live information.

❤️ Save favorite movies
🌙 Dark mode
🎬 Watch movie trailers
📺 Search TV shows too
🎭 Filter movies by genre
